# micro-house
Spring Cloud 打造微服务房产销售平台

#### 启动运行
使用IDEA导入项目

修改配置文件 house-web下的application.properties

启动house-web目录下的HouseApplication.java

启动Nginx（静态图片服务器，图片见nginx文件夹）、Redis服务
